## System Requirements

- Windows / Ubuntu / macOS
- [Node.js](https://nodejs.org/en/)

## How to build
- `npm install`
- `npm run build`

## How to start
- `node build/src/index.js processes.csv` (or any other `processes_file` in the project directory)
